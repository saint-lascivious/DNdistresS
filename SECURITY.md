# Security Policy

## Supported Versions

Security fixes are provided for the latest release on the `master` branch.

## Reporting a Vulnerability

Please **do not** open public issues for suspected vulnerabilities.

Report privately by email to: **saint@sainternet.xyz**

Include, if possible:

- Affected version / commit
- Reproduction steps or proof of concept
- Impact assessment
- Suggested fix (optional)

You will receive an acknowledgment as soon as practical, and updates as triage progresses.
